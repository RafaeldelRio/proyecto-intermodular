Cómo ejecutar una diapositiva:
marp nginx-https.md --pdf --theme beam.css --allow-local-files

Recuerda que para que el preview funcione, debes añadir el css de beam al settings de VSCode.
https://rnd195.github.io/marp-community-themes/theme/beam.html
https://ctan.org/pkg/cm-unicode?lang=en

Las fuentes las debemos importar en linux:
sudo cp -r cm-unicode /usr/local/share/fonts/
